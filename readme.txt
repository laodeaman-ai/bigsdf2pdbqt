(1), open and edit bigsdf2pdbqt.py by using notepad. 
Go to line: pdb2pdbqt_path = "/path/to/AutoDockTools/Utilities24"
Replace with the path of your autodocktools utility
Exp: /home/<user>/MGLTools-1.5.6/MGLToolsPckgs/AutoDockTools/Utilities24

(2) Put youf sdf database file(s) in current directory.

(3) And then execute in terminal:

python bigsdf2pdbqt.py, or

python3 bigsdf2pdbqt.py
